The system utilizes SQL queries to store, retrieve, and manipulate employee records, ensuring accurate calculations for each payroll cycle. It features automated calculations for salaries, taxes, and bonuses based on predefined business rules, making payroll processing faster and error-free. This project is a SQL-based Instagram clone that mimics core functionalities of the popular social media platform. The system uses SQL to manage user data, posts, likes, comments, and follows. It enables the creation of user profiles, the uploading of posts, and interaction through comments and likes, replicating essential features of Instagram's social network.
